# Project Defense Guide

## What to explain first
Start with the business problem: many fashion stores need a structured way to manage products, orders, customers, and promotions online.

## Core technical points
- React frontend for interactivity
- Express API for business logic
- MongoDB for product, user, coupon, and order storage
- JWT for authentication and role-based access control
- Multer for secure image uploads

## Pages to demo
- Home page
- Products page with filters
- Product details with gallery and variants
- Cart and checkout
- Admin dashboard
- Admin product management
- Admin orders and analytics
