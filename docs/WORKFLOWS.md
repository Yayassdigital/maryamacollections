# Workflows

## Customer workflow
1. Browse products
2. Open product details
3. Choose a variant
4. Add to cart or wishlist
5. Checkout with shipping and payment details
6. View order success page and order history

## Admin workflow
1. Sign in as admin
2. Open dashboard
3. Add or edit products
4. Upload product images and define variants
5. Create coupons
6. Update order status and payment state
7. Review analytics
